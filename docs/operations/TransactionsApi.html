txsDecodePost

Decode a transaction (signed or not) from base64-encoded Amino serialized bytes to JSON

URL

https://api.secrettestnet.io/txs/decode

HTTP Method

POST

Response Type

StdTx

Parameters

  • Body: tx Tx_1(Tx_1)

    The tx to decode

Parameters

txsGet

Search transactions by events.

URL

https://api.secrettestnet.io/txs

HTTP Method

GET

Response Type

PaginatedQueryTxs

Parameters

  • Query: message.action String

    transaction events such as 'message.action=send' which results in the following endpoint: 'GET /txs?message.action=send'. note that each module documents its own events. look for xx_events.md in the corresponding cosmos-sdk/docs/spec directory

  • Query: message.sender String

    transaction tags with sender: 'GET /txs?message.action=send&message.sender=secret16xyempempp92x9hyzz9wrgf94r6j9h5f06pxxv'

  • Query: page Integer

    Page number

  • Query: limit Integer

    Maximum number of items per page

  • Query: tx.minheight Integer

    transactions on blocks with height greater or equal this value

  • Query: tx.maxheight Integer

    transactions on blocks with height less than or equal this value

Parameters

txsPost

Broadcast a signed tx to a full node

URL

https://api.secrettestnet.io/txs

HTTP Method

POST

Response Type

BroadcastTxCommitResult

Parameters

  • Body: txBroadcast TxBroadcast(TxBroadcast)

    The tx must be a signed StdTx. The supported broadcast modes include `\"block\"`(return after tx commit), `\"sync\"`(return afer CheckTx) and `\"async\"`(return right away).
