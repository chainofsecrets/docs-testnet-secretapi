govProposalsGet

Query proposals information with parameters

URL

https://api.secrettestnet.io/gov/proposals

HTTP Method

GET

Response Type

List

Parameters

  • Query: voter String

    voter address

  • Query: depositor String

    depositor address

  • Query: status String

    proposal status, valid values can be `\"deposit_period\"`, `\"voting_period\"`, `\"passed\"`, `\"rejected\"`
